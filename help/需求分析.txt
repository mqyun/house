游客表(visitor)
id                  游客id
account             账号
password            密码
name                姓名
phone               联系电话

用户表(user)
id                  用户id
account             账号
password            密码
name                姓名
phone               联系电话

管理员表(admin)
id                  管理员id
account             账号
password            密码
name                姓名

房源表(house)
id                  房源id
type                类型  (出租0、出售1)
diduan              房源地段  (闹市中心区、次繁华生活区、CBD区域、好环境老城区、城乡结合部、交通好郊区)
huxing              房源户型  (平层户型、跃层户型、错层户型、复式户型)
price               房源租金或售价
zuzhutype           租住类型(NULL)  【当房源为出售时该字段为空】  (整租、合租、日租、周租)
chaoxiang            房源朝向  (东、南、西、北)
mianji              房源面积
userid              房源所属用户id
shenhe              审核状态  (0未审核、1审核通过、2审核未通过==>此时可重新提交房源信息)
youkeid             房源租赁或购买游客id(NULL) 【未出售或出租时该字段为空】
hasimg              是否上传图片  (0未上传、1已上传)

房源图片表(houseimg)
id                  房源图片id
houseid             所属房源
url                 服务器存储地址

功能：
01.游客、用户登录注册，管理员登录
02.游客 ==> 放租房源筛选 ==> (地段、户型、租金、租住类型、朝向、面积) ==> 查看详情
03.游客 ==> 出售房源筛选 ==> (地段、户型、总价、朝向、面积) ==> 查看详情
04.用户 ==> 关注房源
05.用户 ==> 放租(出售)房源 ==> 填写房源信息
06.用户 ==> 查看正在放租和出售的房源及其审核状态
07.用户 ==> 查看已经放租和出售的房源
08.用户 ==> 查看关注的房源
09.用户 ==> 修改放租和出售的未审核状态的房源
10.用户 ==> 删除放租和出售的房源
11.管理员 ==> 审核用户提交的放租和出售的房源
